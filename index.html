Metya 60fps optimizer — Privacy Policy

Last updated: July 2, 2026

This Privacy Policy explains how Metya 60 fps optimizer ("we", "our", or "the extension") collects, processes, stores, and shares user data in compliance with the Chrome Web Store Developer Program Policies.

1. Data Collection (Kullanıcı Verisi Toplama)

We respect your privacy and only collect the minimum amount of data required to authenticate active subscriptions, manage licenses, and protect our systems from abuse. We distinguish between transmitted data (sent to our servers) and locally processed data (which never leaves your device):

A. Transmitted Data (Sent to our server/worker)

  • Discord User ID, Username, and Avatar: Collected securely when you authenticate via Discord OAuth to check subscription status and verify membership in our official Discord server.
  • TikTok Username (Optional): Only collected if manually and explicitly entered by the user in the extension's settings interface. This step is optional and can be skipped entirely.
  • Device ID: A randomly generated unique identifier (e.g., "M-XXXXXX") generated locally and sent to our verification server to manage license active sessions and check access rights.

B. Locally Processed Data (Never sent to any server)

  • Local Session Logs: The optimization status, video IDs processed on TikTok Studio, and methods used are processed purely inside your browser to handle UI states. This data is stored locally in your browser storage and is never transmitted to us or any third party.
  • Local Profiles: Account metrics (follower counts, nicknames, etc.) parsed during optimization features are processed strictly client-side to display in-app statistics. They are never stored or transmitted to external servers.

2. Data Processing and Usage (Veri İşleme)

Collected data is processed strictly for the following purposes:

  • To verify access rights, subscription status, and Discord server membership.
  • To manage active license counts and execute anti-abuse / ban controls.
  • To perform local file rendering, transcoding, and optimization settings on TikTok.

3. Data Storage and Retention (Veri Saklama)

  • Local Storage: Discord session data, preferences, and optimization logs are stored on your device using `chrome.storage.local`. This data is automatically and permanently destroyed when you uninstall the extension.
  • Server Storage: Discord IDs, manually linked TikTok usernames, and Device IDs are securely stored in our Cloudflare Worker database. This data is stored only as long as your license/subscription is active, or until you request its deletion.

4. Data Sharing and Disclosure (Veri Paylaşma)

  • No Third-Party Sharing: We do not sell, rent, trade, or share your data with advertisers, third-party trackers, or marketing networks.
  • OAuth Authentications: Discord OAuth credentials are shared only with Discord Inc. during the login process.
  • Download Functionality: When using the video download helper, only the TikTok video ID is sent to the public TikWM API to fetch the media files. No personal identifiers are shared.

5. User Rights and Data Deletion

You can request the deletion of all data associated with your Discord account or Device ID stored on our servers at any time. To request deletion, please contact us via our support channel.

6. Contact & Support

For inquiries, support, or data deletion requests, join our official Discord server: https://discord.gg/D5m2YYNFaS
